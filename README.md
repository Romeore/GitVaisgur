## Introduction

GitVaisgur is an open-source based on .NET to interact git commands using a GUI instead of the command line.

This project is under development - stay tuned ! 

## System Requirements

- Windows 7 and above
- .NET Framework 4.5.2

## Technologies

- WPF
- MaterialDesign
- MvvmLight
